# XD Messenger

Мессенджер с одноразовым ключом: для чата собеседнику нужны только ваш **аккаунт** и **ключ**.

- Создаёте чат → получаете ключ
- Делитесь ключом и аккаунтом
- Собеседник вводит их и подключается
- Сообщения в реальном времени

## Бесплатный хостинг

- **Frontend**: GitHub Pages (бесплатно)
- **Backend**: Supabase (бесплатно)

---

## Быстрый старт

### 1. Supabase (база и реальное время)

1. Зарегистрируйтесь: [supabase.com](https://supabase.com)
2. Создайте новый проект
3. В **SQL Editor** выполните весь скрипт из `supabase-schema.sql`
4. В **Database → Replication** включите Realtime для таблицы `messages` (если нужно)
5. В **Settings → API** скопируйте:
   - **Project URL**
   - **anon public** key

### 2. Клонирование и настройка

Клонируйте репозиторий. Создайте `config.js` (или скопируйте из `config.example.js`) и укажите данные:

```js
window.APP_CONFIG = {
  supabaseUrl: 'https://ваш-проект.supabase.co',
  supabaseAnonKey: 'ваш_anon_key'
};
```

### 3. Локальная проверка

Откройте `index.html` в браузере или запустите:

```bash
npx serve .
```

### 4. Деплой на GitHub Pages

1. Создайте репозиторий на GitHub
2. Загрузите проект (включая `config.js` с вашими данными)
3. **Settings → Pages**:
   - Source: **Deploy from a branch**
   - Branch: `main` (или `master`)
   - Folder: `/ (root)`
   - Save

Сайт будет доступен по адресу:
   `https://ваш-ник.github.io/имя-репозитория/`

## Как пользоваться

1. **Начать** → ввести своё имя (аккаунт)
2. **Создать чат** → получить ключ и аккаунт
3. Поделиться с собеседником: *аккаунт* и *ключ*
4. Собеседник: **Присоединиться** → вводит ваши данные
5. Чат открывается, сообщения приходят в реальном времени
